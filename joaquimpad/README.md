# Code Editor with Monaco

A multi-tab code editor built with Python, PyQt5, and Monaco Editor.

## Features
- Multi-tab interface with datetime-based tab names
- Monaco editor with syntax highlighting and formatting
- Tab pinning to the left
- Status bar showing line, column, selection, and file size
- Navigation history for cursor position
- File explorer panel
- Icons for documents and code files

## Installation

```bash
pip install -r requirements.txt
```

## Running the Application

```bash
python main.py
```
